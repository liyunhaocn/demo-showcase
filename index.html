Daily operation

每日生产流

  1. 选题 每天从 AI/行业资讯里挑 1 个可盈利切口。
  2. 建 Demo 生成前端可运行版本、截图、验证记录和商业化假设。
  3. 入库 用 `npm run add-demo -- --name ...` 追加到 `data/demos.json`。
  4. 发布 每天提交到 GitHub，Cloudflare Pages 自动部署主站，GitHub Pages 保留备份镜像。
